How to Use Pond

Follow these simple steps in order to start using the Pond App!

1. Course Name

This is where you input the corresponding course name to the assignment.

2. Assignment Title

This is where you input the name of the assignment.

3. Maximum Time to Finish

This is where you input the (approximate) most amount of time to finish this assignment.

4. Due Date

This is what sends all the information about the assignment to the student.
